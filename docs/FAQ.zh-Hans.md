# 常见问题

## 隐私与安全

### 我的照片会被上传吗？

不会。Goja 完全在浏览器中运行，照片始终保留在你的设备上——没有服务器、没有上传、不需要注册账号。

### 我的数据安全吗？

安全。Goja 不会收集、存储或传输任何数据，所有处理都在浏览器本地完成。页面启用了严格的内容安全策略 (CSP)，以防止第三方脚本运行。

## 入门

### 如何使用 Goja？

1. 在浏览器中打开 [Goja](https://fingerfly.github.io/goja/)。
2. 将照片拖放到网格区域，或点击该区域选择文件。
3. 拖动照片可调整顺序；右键（桌面端）或长按（移动端）可移除单张照片。
4. 点击齿轮图标打开设置，调整模板、画布尺寸、间距、背景、水印、拍摄日期叠加和文件名。
5. 点击**导出**，选择分享、保存到本机、复制到剪贴板或在新标签页中打开。

### 能把 Goja 安装到手机或电脑上吗？

可以。Goja 是渐进式 Web 应用 (PWA)。在手机上，使用浏览器的"添加到主屏幕"选项。在桌面端 Chrome 或 Edge 中，点击地址栏中的安装图标或从浏览器菜单选择"安装应用"。

### 支持离线使用吗？

支持。首次访问 Goja 后，Service Worker 会缓存所有资源。断网时会显示"离线"横幅，但所有功能仍然可用。

### 支持哪些语言？

Goja 支持 11 种语言：English、简体中文、繁體中文、Deutsch、Nederlands、Español、Italiano、Türkçe、Suomi、日本語、Esperanto。语言会根据浏览器设置自动切换。

## 照片操作

### 如何添加照片？

将图片文件拖放到网格区域即可。也可以点击拖放区域打开文件选择器，选择一张或多张图片。

### 如何调整照片顺序？

桌面端：将照片拖到另一个格子即可交换位置。移动端：使用触摸拖动。还可以使用键盘快捷键：按 Alt+左/右方向键与前一个或后一个格子交换。

### 如何移除单张照片？

桌面端：右键点击照片，从弹出菜单中选择移除。移动端：长按照片触发同一弹出菜单。

### 如何撤销操作？

按 Ctrl+Z（Windows/Linux）或 Cmd+Z（Mac）可撤销布局更改、位置交换和照片移除。

### 支持哪些照片格式？

Goja 支持浏览器所支持的所有图片格式——通常包括 JPEG、PNG、WebP、GIF 和 HEIF/HEIC（Safari 上）。出于安全原因，SVG 文件不被接受。

## 设置与效果

### 如何更换网格布局？

点击齿轮图标打开设置，使用模板选择器从 15 种以上布局中选择（支持 1 至 9 张照片）。可以手动选择模板，也可以让 Goja 根据照片数量自动选择。

### 如何调整画布尺寸？

在设置中修改宽度和高度的值。也可以直接拖动网格上的调整手柄来改变尺寸。

### 如何添加水印？

在设置中启用水印选项，输入文字（自由文字、日期/时间或版权符号），然后调整位置、透明度和字体大小。水印会在网格上实时预览，并出现在导出的图片中。

### 如何在照片上显示拍摄日期？

在设置中启用拍摄日期叠加功能。Goja 会读取每张照片的 EXIF DateTimeOriginal 字段并显示在图片上。你可以设置位置、不透明度和字号，日期格式会跟随浏览器的语言设置。

### 有哪些滤镜可用？

Goja 提供八种滤镜预设：灰度、复古、亮度、对比度、鲜艳、褪色、复古风、模糊。此外还有暗角效果（边缘变暗），可通过滑块调节强度。滤镜和暗角效果都能在网格上实时预览。

### 有哪些比例预设？

导出的比例预设包括：1:1、3:4、4:3、16:9、Instagram、Stories，以及国内社交媒体格式（抖音、小红书、快手、视频号）。

## 导出

### 支持哪些导出格式？

JPEG 和 PNG。可以在导出选项中选择格式。

### 有哪些导出方式？

点击导出后，可以选择以下四种方式：

- **分享** —— 通过 Web Share API 将图片发送到其他应用
- **保存到本机** —— 将图片文件下载到设备
- **复制到剪贴板** —— 将图片复制到剪贴板
- **在新标签页中打开** —— 在浏览器新标签页中查看导出的图片

### 为什么看不到分享按钮？

分享功能使用 Web Share API，并非所有浏览器都支持（例如桌面端 Firefox 及部分较旧浏览器）。如果你的浏览器不支持，该按钮会自动隐藏。请改用保存到本机或复制到剪贴板。

### 能自定义文件名吗？

可以。在设置中可以为导出的图片设置自定义文件名，还可以选择是否在文件名中附加当前日期。
